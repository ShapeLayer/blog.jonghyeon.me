다중접속 기술은 하나의 전송 매체에 연결된 여러 개 단말기가 그 매체에 동시에 접속해 통신할 수 있게 하는 기술이다. 널리 알려진 고전적인 방식에는 시분할 다중접속(TDMA), 주파수분할 다중접속(FDMA), 코드분할 다중접속(CDMA)이 있다.  

## FDMA: 주파수분할 다중접속, Frequency Division Multiple Access

FDMA는 주파수 대역을 여러 개의 부 대역(sub-band)로 나누고, 나눈 대역을 각 사용자에게 독점시키는 방식이다. 각 사용자는 할당된 주파수 대역 안에서 통신할 수 있다.  

이 방식은 간단하고 구현이 쉬워 1세대 이동통신에서 채택한 다중접속 방식이다. 이 방식에서는 각 사용자가 독립된 주파수 대역을 사용하기 때문에 간섭이 적다. 하지만 주파수 자원의 효율적인 활용이 어려울 수 있고, 사용자가 많아질수록 필요한 주파수 대역이 증가하여 전체 시스템이 비효율적이게 될 수 있다.  

## TDMA: 시분할 다중접속, Time Division Multiple Access

TDMA는 시간 대역을 여러 개의 시간 슬롯으로 나누고, 각 사용자에게 특정 시간 슬롯을 할당하는 방식이다. 각 사용자는 할당된 시간 슬롯 동안 통신할 수 있다. 이 시간 슬롯은 네트워크를 사용하는 모든 사용자에게 순차적으로 할당, 반복적으로 순환된다. 사용자는 본인에게 할당된 시간 슬롯만을 사용할 수 있기 때문에, 시간 동기화가 반드시 필요하다.  

이 방식은 FDMA보다 주파수 자원을 효율적으로 활용할 수 있다는 측면을 주목받아, 2세대 이동통신에서 채택되었다. 하지만 시간 슬롯이 고정되어 있기 때문에, 사용자가 많아질수록 각 사용자에게 할당되는 시간 슬롯이 줄어들어 통화 품질이 저하될 수 있다.  

## CDMA: 코드분할 다중접속, Code Division Multiple Access

CDMA는 FDMA나 TDMA와 달리 주파수나 시간 자원을 나누지 않고, 모든 사용자가 전체 주파수 대역과 시간 자원을 동시에 사용하도록 할 수 있다. 사용자는 각자의 고유한 코드를 사용해 신호를 인코딩하고 디코딩한다. FDMA나 TDMA와 달리, CDMA에서는 여러 사용자가 동시에 같은 주파수 대역과 시간 슬롯을 사용할 수 있어, 3세대 이동통신에서 채택되었다.  

CDMA는 스펙트럼 확산 기술을 사용하여 각 사용자의 신호가 다른 사용자들의 신호에 비해 낮은 전력으로 전송되도록 한다. 이를 통해 간섭을 줄이고, 여러 사용자가 동시에 통신할 수 있도록 한다.

이러한 특징으로 높은 스펙트럼 효율을 보이고, 사용자 수용 용량에 있어 유리한 측면이 있다. FDMA나 TDMA에서는 할당 가능한 자원이 모두 소진되면 더 이상 사용자를 수용할 수 없는데, CDMA는 사용자가 늘어나면 통화 품질이 점진적으로 저하될 뿐, 접속이 차단되지는 않는다. 이 특징을 소프트 용량(Soft Capacity)라고 하고, FDMA나 TDMA의 하드 용량(Hard Capacity)과 구분한다.
